MessageBuffer
    结构为rpos,wpos,storage = std::vector<uint8> 

    每个socket都有一个自己提供的buffer供libuv用，这个buffer可以复用，他的生命周期和TcpSocketBase一样长
    当有读事件的时候，libuv会自动把数据填入的这个buffer中
