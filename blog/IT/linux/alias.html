 设置alias

Before beginning, create a file called ~/.bash_alias:

$ touch ~/.bash_alias

Then, make sure that this code appears in your ~/.bashrc file:

if [ -e $HOME/.bash_alias ]; then  
source $HOME/.bash_alias  
fi
