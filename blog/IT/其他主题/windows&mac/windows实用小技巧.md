## 截屏

* Win + PrScrn：复制全屏到内存，使用 Ctrl + V 可粘贴。
* Win + Shift + S；调用Windows自带的“截图和草图”软件进行截屏


## 快捷键
* 打开搜索栏  `单独的win键`
* 新建文件夹    `ctrl + shift + n`
* 关闭窗口       `ctrl + w`
* 打开资源管理器    `ctrl + e`
* 切换窗口  `ctrl + tab`
* 锁定显示屏  `win+ L`
* 最小化所有窗口  `win + d`
* 打开通知中心  `win + A`
