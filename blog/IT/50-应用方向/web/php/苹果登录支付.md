## 苹果退款
苹果在2020年6月的开发者大会上宣告支持普通商品的退款回调服务，只需要在开发者后台配置回调网址即可收到退款通知
苹果给出推送退款的功能，是 server to server，所以我们需要将接收退款通知推送的接口填入苹果后台。
处理退款主要是先判断notification_type是否为REFUND，是的话再根据其他的参数来处理退款订单
