c语言中文网：C语言运算符优先级和结合性一览表
http://c.biancheng.net/view/161.html

```
括号、成员第一;    //括号运算符`[]() . ->`
全体单目第二;      //右到左结合性。 所有的单目运算符比如++、 --、+(正)、-(负)、!、 ~、取值运算符*、&取地址、sizeof、(类型)强制转换
乘除余三,加减四;   //这个"余"是指取余运算即%
移位五，关系六;    //移位运算符：<< >> ，关系：> < >= <= 等
逗号运算级最低!    //逗号运算符优先级最低 
```

左结合
```c
int result = 10 - 5 - 2; 
```

右结合
```c
int a, b, c;
a = b = c = 10;  // 等同于 a = (b = (c = 10))

b = !!b;
```
